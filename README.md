# Predicting Canadian Residential Housing Prices

## 📌 Project Overview
This project focuses on predictive analytics within the Canadian real estate market. Using a dataset of **44,000+ residential listings**, I developed machine learning models to estimate property values and identify the primary factors driving housing prices across the country.

The goal was to provide a data-driven tool that helps stakeholders understand market trends and price expectations with high reliability.

## 🚀 Key Results
*   **Model Performance:** Achieved a **71.79% accuracy score (R-squared)**.
*   **Top Predictors:** Identified key pricing drivers such as location (province/city), square footage, and number of amenities.
*   **Algorithms Compared:** Evaluated Multiple Linear Regression (MLR) against Random Forest to find the most robust predictive power.

## 🛠️ Tech Stack
*   **Language:** Python 3.x
*   **Data Manipulation:** Pandas, NumPy
*   **Machine Learning:** Scikit-Learn (Random Forest Regressor, Linear Regression)
*   **Visualization:** Matplotlib, Seaborn
*   **Environment:** Jupyter Notebook / Python Scripts

## 📊 Methodology
1.  **Data Cleaning:** Handled missing values, removed outliers, and standardized categorical variables (One-Hot Encoding).
2.  **Exploratory Data Analysis (EDA):** Analyzed correlations between property features and final sale prices.
3.  **Feature Engineering:** Selected the most impactful variables to improve model efficiency.
4.  **Modeling:** 
    *   **Multiple Linear Regression:** Used as a baseline model.
    *   **Random Forest:** Implemented to capture non-linear relationships in the data.
5.  **Evaluation:** Measured performance using R-squared, Mean Absolute Error (MAE), and RMSE.
